Usage

1. Set Claude API key:

$ echo "{{CLAUDE_API_KEY}}" > ~/.workback/api_key

2. Navigate to your project:

$ cd path_to_your_project

3. Launch:

$ workback

4. Start with logs or github issues to find the root cause:

$ >>> /analyze error.log
$ >>> /github https://github.com/ParabolInc/parabol/issues/11258

Security

WorkBack runs entirely between you and your private Claude API key. For questions or to report issues, contact founders@hyperdrive.engineering.
